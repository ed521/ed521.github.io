DIKE 禪聲 DS603 多功能木紋藍牙音響，體驗高 CP 值的音樂饗宴！

發表於 2018-12-31 | 分類於 開箱文 | 評論數： 0 | 閱讀次數：
| 文章字數： 2.9k

藍牙喇叭最大的優點就是無線化，手機十分輕鬆就能透過藍牙和喇叭連線，然後播放音樂。這次要體驗的 DIKE DS603 音響功能非常完善，不管是走有線（3.5mm音源接孔）和無線（藍牙）都沒問題，甚至支援外接隨身裝置的媒體播放，只要先將 USB 和 TF 卡存入音樂，再插入 DS603 即可播放音樂。除此之外，DS603 亦搭載時鐘和鬧鐘功能；讓它不僅僅是台喇叭，更是一台用途廣泛的多功能音響。

本次要開箱的產品為 DIKE 禪聲 DS603 - 多功能一體式藍牙音響。

從外盒的介紹可以得知，DS603 的功能實在是非常完備，它具有時鐘顯示和鬧鐘功能，音訊則有藍牙和 AUX 輸入模式兩種；另外還支援 USB 和 MicroSD 卡的音樂播放，並附帶一支迷你遙控器，播歌、調整音量、切換輸入模式等，在遠處就能操控。

接著看到產品內容物，除了 DS603 音響之外，還有遙控器、音源線，以及一份使用說明書。

遙控器的功能如下，可以對音響進行開機、輸入模式切換、音量、曲目等控制；要注意的是，遙控器的電池型號為 CR2025，如果更換錯誤的電池可能會無法使用。

再來是使用說明書，DS603 所有詳盡的功能和使用說明皆附註在內。

最後看到 DS603 音響，採用的是木紋外觀，且四周為圓弧邊緣；正面配置了兩個圓形喇叭，並配上金屬網罩來保護。在喇叭中間則有顯示螢幕，下方有幾個常用的功能按鈕。

在機器後方備有電源開關，還有 3.5 mm 的音源輸入孔和 TF 卡(MicroSD 卡)和 USB 的插孔，以及產品規格、額定電流、製造商等資訊。

音響底部則具備一個大大的低音強化振膜，而四腳採用了矽膠腳墊架高，能減少雜音及反射聲共振問題。


＃＃＃＃＃ DIKE DS603 功能評測 ＃＃＃＃＃

首先看到 DS603 的顯示螢幕下方的按鈕，可進行開機/關機(待機)、輸入模式切換、播放/暫停和調整音量等基本功能；而完整功能只能在遙控器上操作。

以下介紹 DIKE DS603 的各種功能：

「時鐘模式」

第一次開機需要設定時間，設定完後，之後在待機狀態、鬧鈴和藍牙模式中，就會顯示目前的時間。

「鬧鐘模式」

設定完鬧鐘時間後，時間到就會響鈴，並顯示”-AL-“。鬧鈴音量會隨著時間越來越大，按下遙控器上的任何按鈕即可關閉鬧鐘。

「音訊輸入：藍牙模式」

手機和 DS603 完成藍牙配對後，就能播放出手機裡的聲音。

「音訊輸入：AUX 模式」

AUX 模式也就是 3.5mm 音源線輸入模式，只要一頭插入音響背面的 AUX 孔，另一頭插入要播放音樂的裝置即可。

「音樂播放：TF 卡 / USB 模式」

DIKE DS603 也能透過插入媒體裝置來播放音樂，並支援 USB 隨身碟和 TF 卡，也就是我們平常插在手機中的 Micro SD 記憶卡。

更厲害的是，DS603 音響支援的音樂格式非常多，除了最常見的 mp3 格式，還支援了 wma, wav, ape, flac 等音樂格式。

總結以上功能，筆者錄了一段功能介紹的影片：

  1. 時鐘模式
  2. 鬧鐘模式
  3. 藍牙模式
  4. USB / TF 卡模式，並測試上述五種格式檔案，並且支援斷點繼續播放的功能
  5. AUX 模式

＃＃＃＃＃ DIKE DS603 音質評測 ＃＃＃＃＃

DIKE DS603 在音質的表現上也不遜色，筆者拿它與最近在台灣上市的小愛音箱來做比較，其最大的差距在於低頻的表現，因為 DS603 具有一顆低音強化振膜，如此一來就能表現出有一定的下潛量、且有力的低頻，讓整首歌曲的音樂性大幅提升。

筆者也挑了兩首音樂來測試，首先，播放林俊傑的醉赤壁時，DS603 在詮釋人聲上展現了高度的流暢性；林俊傑充滿磁性的嗓音，透過這個音響播放而出，不但有著一定厚度且不失溫暖，背景音樂亦烘托得恰到好處，給聽者一種撫慰人心的深沉情感。

▼ 林俊傑 -【JJ陸】專輯曲目：醉赤壁

第二首歌為 Aimer - Ref:rain。Aimer 向來以渾厚又不失情感的嗓音著名，在副歌的人聲出來時，DS603 播放出了不失 Aimer 特色的爆發力，雖然聲音有些許散掉，但在樂器的分離度來說還不錯，每種樂器的聲音不會被其他樂器蓋過去，大鼓的 punch 聲在音樂中仍然能打動人心，量夠足也潛的下去，整體來說還是挺不賴的。

▼ Aimer -【眩いばかり】專輯曲目：Ref:rain


DIKE DS603 具備鬧鐘功能，並支援待機時的時間顯示。如此一來，在半夜黑漆漆的一片時，也能得知現在的時間。

如果不想要待機時顯示時間，只要在待機狀態下，按一下遙控器上的太陽符號即可關閉。

螢幕顯示器接上 Chromecast，就能在房間內觀看 YouTube、Twitch 等影音資源。此時 DS603 音響有兩種功能，第一種是聲音輸出，第二種是作為 Chromecast 的 USB 供電，如下圖所示。

而 DIKE DS603 也能當作床頭音響，利用藍牙播放手機中的音樂，剛好搭配 KKBOX 內建的定時器，時間到就會自動關閉音樂，安穩地進入夢鄉。

如果把 DS603 放在客廳，透過事先儲存在隨身碟的音樂，它就是一台能自己播放音樂的音響；使用遙控器就可以切換歌曲、暫停/繼續播放，調整音量等，十分方便。另外值得一提的是，每次切換到 TF 卡 / USB 音樂播放模式時，音樂會從上次斷掉的地方繼續播放，而不是從頭開始播。

在辦公室裡，DS603 就是一台電腦的喇叭，我還可以將文件等資料放在音響上方，不占任何空間。


假日早上，我被 DIKE DS603 的鬧鈴聲喚醒，接著使用它來播放隨身碟中的輕音樂，看著小說度過悠閒的早晨。

下午的時候，久違不見的朋友來家裡聚會，相談甚歡；同時使用手機藍牙連結 DIKE 音響，播放炒熱氣氛的音樂。

傍晚，我們利用小型投影機投影畫面到牆壁，並用 AUX 音源線外接喇叭，看了一場精采的球賽直播。

睡覺之前，繼續播放早上未聽完的音樂，並且把剩餘幾頁的小說看完。

由於 DIKE DS603 的功能眾多，從以上的情境來看，生活中的大小事都能和它形影不離。總結 DS603 音響的特色如下：

1. 全方位功能：內建藍牙 / Aux-In / USB 隨身碟 / TF 記憶卡等音訊輸入模式。
2. 全木箱結構：不僅帶來圓潤舒適的中音及高頻，更顯禪意氣息。
3. 低音強化振膜：使用下點火式的低音振模，強化低音表現。
4. 時間守護者：顯示屏幕可設定時間外，也能設定鬧鈴喚醒美好早晨。
5. 減震矽膠角錐：特別訂製減震矽膠角椎，減少雜音及反射聲共振問題。
6. 專業音頻芯片：精確的聲音分頻技術，將每一個細節演繹的更加完美。

綜觀以上所述，DIKE DS603 真的是一款 CP 值高到爆表的多功能音響！
